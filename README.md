# write_excel 使用文档

## 1. 软件介绍

该软件用于整合各sheet表到excel输出，支持格式化输出和条件过滤

## 2. 安装说明

无需安装，直接调用 write_xlsx.py 文件即可

## 3. 使用说明

#### 1. 参数列表

|参数|含义|
|-|-|
|-f|指定输出格式配置文件，不指定则全部使用默认格式|
|-o|输出的excel文件名|
|-r|指定过滤规则配置文件，不指定则不进行过滤|
|-b|读取sheet表的输入目录|
|-s|样本名|
|-i|sheet表名和对应的文件路径，路径为相对`sheet表的输入目录`的路径, 格式参见`格式说明`|
